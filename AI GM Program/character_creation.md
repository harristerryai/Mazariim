## Overview

Create a fully detailed Cypher System player character following the house rules in the CypherSystem folder. Player characters are different from NPCs, heroes, and villains in that they have stats like Edge, Effort, and skill progression.

**Save Location:** All character sheets are saved to:

    ~/Desktop/CypherSystem/Setting - Fantasy/Characters/

**File Naming Convention:**

    [CharacterName]_[Type]_Tier[X].md

Examples: Vaelthrix_Stormcrown_Mage_Tier2.md, Aldric_Stoneshield_Paladin_Tier3.md

## Character Creation Steps

### 1. Character Concept

Determine the character sentence: **"I am a [Descriptor] [Ancestry] [Type] who [Focus]"**

Fantasy characters include Ancestry in the character sentence, placed between Descriptor and Type.

- Choose a **Descriptor** from the Cypher System Core Rulebook such as Tough, Swift, Intelligent, Charming, and so on
- Choose an appropriate **Character Type** from the ten available types: Barbarian, Bard, Druid, Fighter, Magus, Mage, Paladin, Priest, Ranger, or Rogue
- Choose an **Ancestry** from the standard options of Dwarf, Elf, Gnome, Halfling, or Human, or from nonstandard ancestries such as Avian, Celestial, Draconic, Elemental, Feline, Goblinoid, Golem, or Infernal
- Choose a **Focus** from the Cypher System Core Rulebook or from rules_foci.md for additional fantasy-specific foci including Warlock Pacts, Sorcerer Bloodlines, Druid Circles, Wizard Schools, and Cleric Domains
- Optionally choose a **Flavor** from rules_flavors.md to simulate multiclassing or specialized training, choosing from Combat, Magic, Stealth, or Skills and Knowledge
- Identify the character's **tier level** from 1 through 6

### 2. Stat Pools

Start with base stats from the chosen Character Type. Each type has different starting pools.

| Type | Might | Speed | Intellect |
|---|---|---|---|
| Barbarian | 12 | 10 | 7 |
| Bard | 8 | 9 | 12 |
| Druid | 8 | 9 | 12 |
| Fighter | 10 | 10 | 8 |
| Magus | 9 | 10 | 10 |
| Mage | 7 | 9 | 12 |
| Paladin | 10 | 9 | 10 |
| Priest | 8 | 9 | 12 |
| Ranger | 10 | 10 | 9 |
| Rogue | 9 | 11 | 8 |

Always check the character type file for exact starting pools as the table above is a quick reference.

Then apply the following additions in order:

- Add **6 bonus points** distributed among pools at character creation (Tier 1)
- Add **4 points per tier advancement** at Tier 2, 3, 4, 5, and 6
- Add any **ancestry bonuses** such as Human receiving +4 to one pool
- Include any **stat pool bonuses from Enabler abilities** such as Enhanced Intellect, Greater Enhanced Intellect, Enhanced Might, Enhanced Speed, Enhanced Potential, and Greater Enhanced Potential

### 3. Edge

- Start with Edge from the Character Type, which varies by type. Check each type's starting Edge options.
- Add any **ancestry bonuses** such as Human receiving +1 Edge to one stat that is currently 0
- Add **+1 Edge per tier advancement** at each tier after the first. The player can choose which stat receives the Edge each time.
- Show final Edge values for Might, Speed, and Intellect

### 4. Effort

Effort equals the character's tier. Tier 1 equals 1 Effort, Tier 2 equals 2 Effort, and so on up to Tier 6 equaling 6 Effort. Note any Effort advantages from character type abilities or other sources.

### 5. Skills

**Critical Rules:**

Start with base skills from the Character Type. The number varies by type: Fighter gets 3, Rogue gets 4, Bard gets 4, and so on. Add any ancestry skill bonuses such as Elf being trained in Perception or Human being trained in two skills of choice. Gain +1 skill advancement per tier at Tier 2, 3, 4, 5, and 6.

**Skill Progression:**

- **Trained** (available from Tier 1): Eases the task by 1 step, which is -3 to the target number
- **Specialized** (available from Tier 3): Eases the task by 2 steps, which is -6 to the target number. Requires the skill to have been taken a second time.
- **Expert** (available from Tier 5): Eases the task by 3 steps, which is -9 to the target number. Requires the skill to have been taken a third time.

**Skills vs Class Features — Do Not Confuse:**

- **Attacks** cannot be improved via regular skills. They can only be improved through class features like Skill with Attacks or Mastery with Attacks.
- **Defense** cannot be improved via regular skills. It can only be improved through class features like Skill with Defense or Mastery with Defense.
- **Initiative** cannot be improved via regular skills. It uses standard Speed Pool rolls.
- **Spell Focus** covering schools like Fire or Necromancy is a class feature, not a regular skill.

**Complete Regular Skills List:**

Acrobatics, Animal Handling, Appraise, Athletics, Craft (specify type such as alchemy, armor, or weapons), Deception, Heal, Insight, Intimidation, Investigation, Knowledge (specify such as History, Local, Medicine, Nature, Nobility, or Religion), Language (specify language), Navigation, Perception, Performance, Persuasion, Profession (specify profession), Research, Sleight of Hand, Spellcraft, Stealth, Survival, and Vehicle (specify vehicle type).

Each character type has a specific list of available skills. Always check the Type file for which skills that class can choose from. For example:

- **Paladin** can choose from: Animal Handling, Athletics, Craft, Insight, Language, Knowledge (History, Religion, Nobility, Local, Medicine), Navigation, Perception, Persuasion, Profession, Spellcraft, and Vehicle
- **Fighter** can choose from: Acrobatics, Animal Handling, Athletics, Craft, Intimidation, Language, Perception, Profession, Stealth, Survival, and Vehicle
- **Mage** can choose from: Appraise, Craft, Insight, Investigation, Knowledge, Language, Navigation, Perception, Profession, Research, Spellcraft, and Vehicle
- **Bard** can choose any four skills with no restrictions

### 6. Ancestry Abilities

List all abilities granted by the chosen ancestry from rules_racesfantasy.md. These are in addition to class abilities and are typically Enablers.

**Examples:**

- **Dwarf** receives Darkvision, Dwarven Resilience, Dwarven Combat Training, and Underground
- **Elf** receives Low-Light Vision, Keen Senses, Fey Ancestry, and Elven Weapon Training
- **Human** receives +4 Pool Points to one pool, Improved Edge of +1 to one stat at 0, and Skilled granting training in two skills

### 7. Powers, Abilities, and Spells

At Tier 1, the number of abilities varies by type, typically 4 to 5. Check the specific character type's Tier 1 section for the exact number. Spellcasters such as Mage, Priest, Druid, and Bard have spell slots in addition to or instead of standard abilities. Each tier after 1 grants additional powers and spells as specified by the type.

For each power or ability, list the following:

- **Name**
- **Cost** in stat pool points, if any
- **Stat Pool** that it draws from, being Might, Speed, or Intellect
- **Range** such as Immediate, Short, Long, or Very Long
- **Duration** if applicable
- **Description** including the full text of what the power does, including damage, effects, and any special rules

Do not forget Enabler abilities that add to stat pools such as Enhanced Intellect, Enhanced Might, Enhanced Speed, and similar permanent bonuses.

### 8. Class Features by Tier

Each Character Type has different class features at each tier. Always reference the specific type's tier advancement table from the Types folder.

**Example — Fighter Type:**

- **Tier 1:** Edge choice of Might or Speed 1, Effort 1, Weapon and Armor Proficiencies for all types, Martial Skills choosing Skill with Attacks or Skill with Defense, and 4 Special Abilities
- **Tier 2:** Martial Skills choosing from Skill with Attacks, Skill with Defense, or Experienced in Armor, and 2 Special Abilities
- **Tier 3:** Artifact Use increases to 3, Extra Attack gained, and 3 Special Abilities
- **Tier 4:** Martial Skills choosing from Mastery or other options, and 2 Special Abilities
- **Tier 5:** Artifact Use increases to 4, Martial Skills choosing from Mastery or other options, and 3 Special Abilities
- **Tier 6:** Triple Attack gained, and 2 Special Abilities

**Example — Mage Type:**

- **Tier 1:** Intellect Edge 1, Effort 1, Artifact Use 3, Magical Training, Spellcasting, 4 Spell Slots, Spellbook with 6 low-tier spells, Spellcasting Skills choosing Spell Focus or Skill with Defense, and Cantrip plus 3 other Special Abilities
- **Tier 2:** Artifact Use increases to 4, 2 additional Spell Slots for 6 total, 3 low-tier spells added to spellbook, Spellcasting Skills choice, and 1 Special Ability
- **Tier 3:** 2 additional Spell Slots for 8 total, 3 low or mid-tier spells added to spellbook, Spellcasting Skills choice, and 2 Special Abilities
- **Tier 4:** Artifact Use increases to 5, 2 additional Spell Slots for 10 total, 3 low or mid-tier spells added to spellbook, Spellcasting Skills choice, Martial Skills choice, and 1 Special Ability
- **Tier 5:** 2 additional Spell Slots for 12 total, 3 spells of any tier added to spellbook, Spellcasting Skills choice, Martial Skills choice, and 2 Special Abilities
- **Tier 6:** Artifact Use increases to 6, 2 additional Spell Slots for 14 total, 3 spells of any tier added to spellbook, Spellcasting Skills choice, Martial Skills choice, and 1 Special Ability

Each character type has unique progression. Always document the specific type's features from the type file.

### 9. Combat Modifiers Summary

Create a clear summary section showing all combat modifiers:

- **Attack Modifiers** listing all class features that ease attacks, which are not regular skills
- **Defense Modifiers** listing all class features that ease defense, which are not regular skills
- **Spell-Specific Modifiers** listing Spell Focus bonuses for relevant spells
- **Ancestry Combat Modifiers** listing weapon training or combat advantages from ancestry
- Show total steps of easing for common actions

### 10. Equipment and Notes

- List appropriate equipment for the character
- Note weapon and armor training from the character type, which varies by type
- Note weapon and armor training from ancestry such as Dwarven Combat Training or Elven Weapon Training
- Include tactical notes about how to play the character effectively
- Note any restrictions such as armor interfering with spellcasting for Mages and other arcane casters
- Include spellbook contents for spellcasters with all known and prepared spells listed
- Note gold pieces and starting wealth

## Common Mistakes to Avoid

- **Forgetting +4 stat points per tier.** Characters gain 4 points to distribute each time they advance a tier.
- **Forgetting +1 skill per tier.** Characters gain one skill advancement each tier after the first.
- **Treating attacks and defense as regular skills.** These can only be improved through class features, not regular skill selections.
- **Not distinguishing class features from skills.** Spell Focus, Skill with Attacks, and similar abilities are class features that come at specific tiers, not regular skills.
- **Assuming all types have the same class features.** Each character type has unique tier progression. Fighter gets Extra Attack at Tier 3 and Triple Attack at Tier 6. Mage gets more spell slots. Paladin gets Extra Attack at Tier 4. Each type is different.
- **Forgetting skill progression rules.** Skills can only become Specialized at Tier 3 or higher and Expert at Tier 5 or higher.
- **Forgetting Enabler abilities in stat pools.** Enhanced Intellect and similar abilities add permanent points to pools that must be included in the maximum.
- **Not checking starting skills.** Different types start with different numbers of skills. Rogue gets 4 and most others get 3.
- **Forgetting ancestry abilities.** All ancestries grant abilities that must be included on the character sheet.
- **Not including Ancestry in the character sentence.** Fantasy characters use the format: I am a [Descriptor] [Ancestry] [Type] who [Focus].
- **Forgetting spell slots and spellbooks.** Spellcasters have specific spell management systems with preparation rules and slot counts.
- **Mixing up spell slots and abilities.** Some classes have both spell slots and special abilities. Some have only one or the other. Check each type carefully.
- **Forgetting artifact attunement limits.** Each type has a specific maximum number of artifacts they can attune to at each tier. This is different from cyphers which have no carry limit.
- **Forgetting armor restrictions for spellcasters.** Mages cannot cast spells in any armor. Priests and Druids cannot cast in armor they are not proficient with. Check each type's restrictions.

## Tone and Approach

- Informative and precise
- Audience is a 40-year veteran gamemaster
- Always cite exact rule text when needed
- Never make up rules. Ask for clarification if uncertain.
- Double-check all calculations

- Cross-reference spell descriptions from the Spells folder and the Spell Compendium
- Include ancestry name in the character sentence between Descriptor and Type

# Character Sheet Format Template

Every character sheet contains two sections. Section 1 is the CHARACTER SHEET, the player-facing reference used during play. It contains all mechanical information needed to run the character without referencing external documents. All ability descriptions include full text. Section 2 is the CHARACTER DETAILS AND CALCULATIONS, the builder and GM reference showing exactly how every number was derived, every choice made, and the rationale behind each decision.

## Formatting Rules

- All files are saved as markdown (.md) files
- All ability descriptions must include full rules text, not summaries or abbreviations
- All pool costs must show both base cost and cost after Edge or Spell Focus reductions
- All skills must note their source in parentheses after the skill name
- Class features (Spell Focus, Skill with Attacks, Skill with Defense, etc.) are listed separately from regular skills but appear
