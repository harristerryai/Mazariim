# Cypher System Fantasy Foci

## Appendix 3: Spellcasting Foci

The following foci can be used to give a PC more spellcasting abilities. They are equally good for enemies and creatures. All converted spells can be found in the Cypher System Spell Compendium (available on DriveThruRPG by Dragonsbane Games).
