# Cypher System Fantasy Player Character Creation Prompt

## Overview
Create a fully detailed Cypher System player character following the house rules in the AI Drive CypherSystem folder. Player characters are different from NPCs, heroes, and villains in that they have stats like Edge, Effort, and skill progression.

**Default Save Location**: Characters created will be saved to `/CypherSystem/Setting - Fantasy/Characters/` folder by default.

## Character Creation Steps

### 1. Character Concept
Determine character sentence: **"I am a(n) [Descriptor] [Ancestry] [Type] who [Focus]"**

**Note**: Fantasy characters include Ancestry in the character sentence, placed between Descriptor and Type.

- Choose **Descriptor** from the Cypher System Core Rulebook (e.g., Tough, Swift, Intelligent, Charming, etc.)
- Choose appropriate **Character Type** (Barbarian, Bard, Druid, Fighter, Magus, Paladin, Priest, Ranger, Rogue, Wizard)
- Choose **Ancestry** (Dwarf, Elf, Gnome, Halfling, Human, or nonstandard ancestries like Avian, Celestial, Draconic, Elemental, Feline, Goblinoid, Golem, Infernal)
- Choose **Focus** from the Cypher System Core Rulebook or **rules_foci.md** for additional fantasy-specific foci
- **Optional: Choose a Flavor** from **rules_flavors_fantasy.md** (Combat, Magic, Stealth, or Skills and Knowledge) to simulate multiclassing or specialized training
- Identify the character's **tier level** (1-6)

### 2. Stat Pools
Start with base stats from chosen Character Type - Each type has different starting pools:

- **Barbarian**: Might 12, Speed 10, Intellect 7
- **Bard**: Might 8, Speed 9, Intellect 12
- **Druid**: Might 8, Speed 9, Intellect 12
- **Fighter**: Might 10, Speed 10, Intellect 8
- **Magus**: Might 9, Speed 10, Intellect 10
- **Paladin**: Might 10, Speed 9, Intellect 10
- **Priest**: Might 8, Speed 9, Intellect 12
- **Ranger**: Might 10, Speed 10, Intellect 9
- **Rogue**: Might 9, Speed 11, Intellect 8
- **Wizard**: Might 7, Speed 9, Intellect 12

**Always check the character type for exact starting pools**

- Add **6 bonus points** distributed at character creation (Tier 1)
- Add **+4 points per tier advancement** (Tier 2, 3, 4, etc.)
- Add any **ancestry bonuses** (e.g., Human: +4 to one pool)
- Include any **stat pool bonuses from Enabler abilities** (Enhanced Intellect, Greater Enhanced Intellect, etc.)

### 3. Edge
- Start with Edge from Character Type (varies by type - check each type's starting Edge)
- Add any **ancestry bonuses** (e.g., Human: +1 Edge to one stat)
- Add **+1 Edge per tier advancement** (can choose which stat each time)
- Show final Edge values for Might, Speed, and Intellect

### 4. Effort
- Effort equals character tier (Tier 1 = 1 Effort, Tier 4 = 4 Effort, etc.)
- Note any Effort advantages from character type (varies by class)

### 5. Skills
**CRITICAL RULES:**

- Start with base skills from Character Type (number varies: Fighter gets 3, Rogue gets 4, etc.)
- Add any **ancestry skill bonuses** (e.g., Elf: trained in Perception; Human: trained in two skills of choice)
- Gain **+1 skill advancement per tier** (Tier 2, 3, 4, etc.)

**Skill Progression:**
- **Trained** (Tier 1+): -1 step to task difficulty
- **Specialized** (Tier 3+): -2 steps to task difficulty (requires taking skill a second time)
- **Expert** (Tier 5+): -3 steps to task difficulty (requires taking skill a third time)

**Skills vs Class Features - DO NOT CONFUSE:**
- **ATTACKS** cannot be improved via regular skills - only through class features like "Skill with Attacks" or "Mastery with Attacks"
- **DEFENSE** cannot be improved via regular skills - only through class features like "Skill with Defense" or "Mastery with Defense"
- **INITIATIVE** cannot be improved via regular skills - it uses standard Speed Pool rolls
- **Spell Focus** (Fire, Necromancy, etc.) is a class feature, not a regular skill
- **Complete Regular Skills List:**
- Acrobatics
- Animal Handling
- Appraise
- Athletics
- Craft (specify type: alchemy, armor, weapons, etc.)
- Deception
- Heal
- Insight
- Intimidation
- Investigation
- Knowledge (specify: History, Local, Medicine, Nature, Nobility, Religion, or other)
- Language (specify language)
- Navigation
- Perception
- Performance
- Persuasion
- Profession (specify profession)
- Research
- Sleight of Hand
- Spellcraft
- Stealth
- Survival
- Vehicle (specify vehicle type)

**NOTE:** Each character type has a specific list of available skills - always check the Type file for which skills that class can choose from. For example:
- **Paladin**: Animal Handling, Athletics, Craft, Insight, Language, Knowledge (History, Religion, Nobility, Local, Medicine), Navigation, Perception, Persuasion, Profession, Spellcraft, Vehicle
- **Fighter**: Acrobatics, Animal Handling, Athletics, Craft, Intimidation, Language, Perception, Profession, Stealth, Survival, Vehicle
- **Wizard**: Appraise, Craft, Insight, Investigation, Knowledge, Language, Navigation, Perception, Profession, Research, Spellcraft, Vehicle
- **Bard**: Can choose ANY four skills (no restrictions)

### 6. Ancestry Abilities
List all abilities granted by the chosen ancestry from **rules_racesfantasy.md**. These are in addition to class abilities and are typically Enablers.

**Examples:**
- **Dwarf**: Darkvision, Dwarven Resilience, Dwarven Combat Training, Underground
- **Elf**: Low-Light Vision, Keen Senses, Fey Ancestry, Elven Weapon Training
- **Human**: +4 Pool Points, +1 Improved Edge, +2 Skilled

### 7. Powers/Abilities/Spells
**Tier 1**: Number varies by type (typically 4-5 abilities)
- Check the specific character type's Tier 1 section for exact number
- Spellcasters (Wizard, Priest, Druid, Bard) have spell slots instead of or in addition to abilities
- Each tier after 1: Gain additional powers/spells (varies by type)

For each power, list:
- **Name**
- **Cost**: X [Stat] points
- **Stat Pool**: Which stat it draws from (Might, Speed, or Intellect)
- **Range**: Immediate, Short, Long, etc.
- **Duration**: If applicable
- **Description**: Full text of what the power does, including damage, effects, etc.

Don't forget **Enabler abilities** that add to stat pools (Enhanced Intellect, etc.)

### 8. Class Features by Tier
**IMPORTANT**: Each Character Type has different class features at each tier. Always reference the specific type's tier advancement table from the **Types folder**.

**Example - Fighter Type:**

- **Tier 1**: Edge (Might or Speed 1), Effort (1), Weapon/Armor Proficiencies, Martial Skills (choose Skill with Attacks OR Defense), Abilities (4)
- **Tier 2**: Edge increase (+1), Martial Skills (choose from list), Abilities (+2)
- **Tier 3**: Artifact Use (3), Extra Attack, Abilities (+3)
- **Tier 4**: Martial Skills (choose Mastery option or other), Abilities (+2)
- **Tier 5**: Artifact Use (4), Martial Skills (choose Mastery option or other), Abilities (+3)
- **Tier 6**: Triple Attack, Abilities (+2)

**Example - Wizard Type:**

- **Tier 1**: Edge (Intellect 1), Effort (1), Artifact Use (3), Magical Training, Spellcasting, Spell Slots (4), Spellbook (6 low-tier spells), Spellcasting Skills (choose Spell Focus OR Skill with Defense), Abilities (Cantrip + 3 others)
- **Tier 2**: Edge increase (+1), Spell Slots (+2), Add spells to spellbook (2), Abilities (+2)
- **Tier 3**: Edge increase (+1), Artifact Use (4), Spell Slots (+2), Add spells to spellbook (2), Abilities (+3)
- **Tier 4**: Edge increase (+1), Spell Slots (+2), Add spells to spellbook (2), Abilities (+2)
- **Tier 5**: Edge increase (+1), Artifact Use (5), Spell Slots (+1), Add spells to spellbook (1), Abilities (+3)
- **Tier 6**: Edge increase (+1), Spell Slots (+1), Add spells to spellbook (1), Abilities (+2)

Each character type has unique progression - always document the specific type's features.

### 9. Combat Modifiers Summary
Create a clear summary section showing all combat modifiers:

- **Attack Modifiers**: List all class features that ease attacks (not regular skills)
- **Defense Modifiers**: List all class features that ease defense (not regular skills)
- **Spell-Specific Modifiers**: Spell Focus bonuses for relevant spells
- **Ancestry Combat Modifiers**: Weapon training or combat advantages from ancestry
- Show total steps of easing for common actions

### 10. Equipment & Notes
- List appropriate equipment for the character
- Note weapon/armor training from character type (varies by type)
- Note weapon/armor training from ancestry (e.g., Dwarven Combat Training, Elven Weapon Training)
- Include tactical notes about how to play the character effectively
- Note any restrictions (e.g., armor interfering with spellcasting for Wizards)
- Include spellbook contents for spellcasters

## Common Mistakes to Avoid

- **Forgetting +4 stat points per tier** - Characters gain 4 points to distribute each time they advance a tier
- **Forgetting +1 skill per tier** - Characters gain one skill advancement each tier
- **Treating attacks/defense as regular skills** - These can ONLY be improved through class features, not regular skill selections
- **Not distinguishing class features from skills** - Spell Focus, Skill with Attacks, etc. are class features that come automatically at certain tiers
- **Assuming all types have same class features** - Each character type has unique tier progression (Fighter gets Extra Attack at Tier 3, Wizard gets more spell slots, etc.)
- **Forgetting skill progression rules** - Skills can only become Specialized at Tier 3+, Expert at Tier 5+
- **Forgetting Enabler abilities in stat pools** - Enhanced Intellect and similar abilities add permanent points to pools
- **Not checking starting skills** - Different types start with different numbers of skills (Rogue gets 4, most others get 3)
- **Forgetting ancestry abilities** - All ancestries grant abilities that must be included
- **Not including Ancestry in character sentence** - Fantasy characters use: "I am a [Descriptor] [Ancestry] [Type] who [Focus]"
- **Forgetting spell slots and spellbooks** - Spellcasters have specific spell management systems
- **Mixing up spell slots vs abilities** - Some classes have both, some have only one

- Never make up rules - ask for clarification if uncertain
- Double-check all calculations
